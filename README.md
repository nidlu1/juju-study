# 스프링 부트와 AWS로 혼자 구현하는 웹 서비스

## 설명
- 스프링 부트와 AWS로 혼자 구현하는 웹 서비스 책을 통해 공부한 내용들

## 개발환경

- OS : window10
- IDE : IntelliJ
- 언어  : Java
- 프레임워크 : 스프링부트
- 템플릿엔진 : mustache

## 서버 기술
- 호스팅업체 : Amazon

## 1인개발시 유용한 API
- 댓글: Disqus, LiveRe
- 외부서비스 자동연동: Zapier, IFTTT
- 방문자 분석: 구글 애널리틱스
- CDN(트래픽분산) : 클라우드플레어
- 이메일 마케팅: Mailchimp
